📖 IPv6 – Theorie

Was ist IPv6 und warum?

❓ Warum IPv6?

IPv4 hat nur ~4,3 Milliarden Adressen (32 Bit). Diese sind seit 2011 erschöpft. IPv6 löst dieses Problem mit 128 Bit → 340 Sextillionen Adressen.

📊 IPv4 vs IPv6

IPv4: 32 Bit, Dezimal, z.B. 192.168.1.1

IPv6: 128 Bit, Hexadezimal, z.B. 2001:db8::1

Kein Broadcast in IPv6 – nur Multicast & Anycast!

Adressaufbau

📐 Struktur einer IPv6-Adresse 128 Bit, aufgeteilt in 8 Gruppen zu je 16 Bit (= 4 Hex-Zeichen), getrennt durch Doppelpunkte:

2001:0db8:0000:0042:0000:8a2e:0370:7334
→ 8 Gruppen × 16 Bit = 128 Bit

Kürzungsregeln

Regel 1 – Führende Nullen weglassen

0042 → 42

0000 → 0

0001 → 1

Gilt pro Gruppe, nicht übergreifend.

Regel 2 – :: für aufeinanderfolgende Nullgruppen

2001:0db8:0000:0000:0000:0001

→ 2001:db8::1

:: darf nur einmal in einer Adresse vorkommen!

⚠️ Fehler: :: zweimal verwenden! 2001::db8::1 ist UNGÜLTIG – unklar, wie viele Nullen ersetzt werden. Immer nur einmal :: pro Adresse!

Besondere Eigenschaften von IPv6

🔄 SLAAC

Stateless Address Autoconfiguration – Gerät generiert automatisch eine IPv6-Adresse aus dem Netzpräfix + MAC-Adresse (EUI-64). Kein DHCP nötig!

🔍 NDP statt ARP

Neighbor Discovery Protocol ersetzt ARP. Nutzt ICMPv6-Nachrichten: Neighbor Solicitation & Neighbor Advertisement.

🔒 IPsec

In IPv6 standardmäßig unterstützt (war optional in IPv4). Ermöglicht verschlüsselte und authentifizierte Kommunikation.

📦 Kein NAT

Durch die enorme Adresszahl braucht IPv6 kein NAT. Jedes Gerät bekommt eine globale Adresse.

🔁 Kein Broadcast

IPv6 hat keinen Broadcast. Stattdessen: Multicast (ff02::1 = alle Knoten, ff02::2 = alle Router).

⚡ Effizienterer Header

IPv6-Header ist vereinfacht (40 Byte fix). Keine Checksumme, keine Fragmentierung durch Router – schnellere Weiterleitung.

Präfix-Notation (CIDR)

Wie bei IPv4 gibt die Präfixlänge an, wie viele Bits das Netz sind:

Beispiel: 2001:db8:1::/48
→ Die ersten 48 Bit sind der Netzanteil → 80 Bit bleiben für Subnetze & Hosts
→ /64 ist der Standard für ein LAN-Subnetz (64 Bit Netz + 64 Bit Interface-ID)
